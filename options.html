Res hit med SL

About

"Res hit med SL" makes finding your way through Stockholm using Stockholms Lokaltrafik (SL) easier. The easiest way to use it is to select your address bar (Omnibox) and type "sl" and then space. Then you can type where you want to go and press enter. This application has not been approved of by Stockholms Lokaltrafik and is not supported by them in any way, shape or form. If you have any questions about this extension, please email me or reach me at Twitter @johanhil. The source code is available at github.com/johanhil/reshitmedsl.

Usage (right click / context)

  1. Select an address or station on any web site you are browsing.
  2. Right click.
  3. Choose "Res hit med SL" in the menu.
  4. Done! SL.se opens up in a new tab with values already filled out and maybe an itinerary.

Usage (omnibox / address bar)

  1. Type "sl" and then a space and then the station you would like to go to.
  2. Press Enter.
  3. Done! SL.se opens up in the same tab with your itinerary :-)
